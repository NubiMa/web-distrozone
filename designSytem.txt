STREETWEAR DISTRO – DESIGN SYSTEM (MODERN GRADIENT UI)
=====================================================

[ COLOR PALETTE ]
----------------
Primary (Base Black)         : #0F0F0F
Background (Main White)      : #FFFFFF
Background Secondary         : #F5F5F5
Accent (Street Orange)       : #FF6B00
Accent Light                 : #FF9F1C
Text Primary                 : #111111
Text Secondary               : #9CA3AF
Border / Divider             : #E5E7EB


[ GRADIENT SYSTEM ]
-----------------
Primary Gradient             : #FF6B00 → #FF9F1C
Secondary Gradient           : #0F0F0F → #1F2933

Usage Rules:
- Gradient usage             : Allowed (controlled)
- Max gradient per section   : 1
- Gradient placement         : Hero, Promo Banner, Primary CTA
- No gradient on body text
- Product cards remain flat


[ GLOBAL RULE ]
-------------
- Dominant color             : White (65–70%)
- Base black usage           : Header, Footer, Buttons
- Accent usage               : CTA, Price, Badge ONLY
- Max accent color           : 1
- Layout style               : Clean, bento-box inspired
- White space                : Generous
- Neon colors                : Not allowed
